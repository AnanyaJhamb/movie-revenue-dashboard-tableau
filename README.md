## Movie Revenue Dashboard

## Project Overview
Interactive Tableau dashboard analyzing box office performance across $400B+ revenue and 150K+ films, uncovering insights about studio performance, release timing, and ratings impact on revenue.

## Key Results

| Metric | Insight |
|--------|---------|
| Total Revenue Analyzed | **$400B+** |
| Films Analyzed | **150K+** |
| Studio Comparison | Major Studios outperform Independents by **50x** |
| Best Release Window | Summer/Holiday releases yield **3x higher returns** |
| Rating Impact | Films with **8+ ratings** generate highest ROI |

## Tech Stack
- **Visualization:** Tableau
- **Data Processing:** Excel, Python (Pandas)
- **Analysis:** Exploratory Data Analysis, Trend Analysis
